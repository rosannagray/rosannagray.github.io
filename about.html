About

Hello my name is Rosanna. I am a web designer with a keen interest in user interface and user experience design, front-end development, and collaborating with others to build rich applications.

I believe in continuous improvement and am always seeking opportunities to improve myself professionally and personally.

Designer

I have skills in:

  • Responsive web design
  • Visual design
  • User interaction design
  • Creating wireframes in myBalsamiq
  • Designing in Bohemian Sketch
  • Creating and following style guides
  • HTML5, CSS3, and JavaScript
  • Version control with Git
  • Using browser web developer tools
  • Vertical rhythm
  • Using preprocessors such as SASS
  • Google Analytics
  • Using web templates like Handlebars and Liquid
  • Web accessibility and knowldege of the New Zealand Government Web Standards
  • Front-end Frameworks e.g. Twitter Bootstrap and Zurb Foundation
  • Cross-browser testing
  • Using Adobe Illustrator and Photoshop.

I also like to upskill and try out new tools in my spare time.

Creative thinker

No matter when and where, I always find inspiration in something.

My inspirations can sometimes lead to a cool idea. From there, it's just a matter of how I can translate my idea into something real.
