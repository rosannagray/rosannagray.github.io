Interactive Experiments

  • Poppin'Corn

    An application developed for the Nexus 7 for making digital popcorn. The aim of the app is to pop the corn kernels by touching the screen of the device.

  • Wind Chimes

    An application developed for the Nexus 7 for relaxation and zen, using the device's microphone. Blow into the microphone to hear some wind chimes.

2D Graphics

  • Algorithmic Flora

    A 2D image generated using fractals and Python Processing. This piece was selected to be showcased in the 2012 Victoria University's Falculty of Architecture and Design booklet.

3D Graphics

  • Generated Environment - Argia

    An environment generated in Maya using Python script. The envrionment is my interpretation of Argia from Italo Calvino's Invisible Cities.

  • Cyborg of Space

    Inspired by Patricia Piccinini's projects - Nest (2006) and Foundling (2008), this is a concept to combine human and machine features into a new form.

  • 3D Models

    My earlier work in 3D modelling. These models were created using 3Ds Max. The centre model was later printed out in a 3D printer.
