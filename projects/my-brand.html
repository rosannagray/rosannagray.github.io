About the project

My brand development

By redesigning my brand and portfolio from time to time, I could use it as an opportunity to experiment with new web tools, frameworks and languages. It is also a way for me to practice and improve my design and development skills.

The challenge

Creating a brand for myself is challenging. I have to think about what best represents my identity each time I redesign my portfolio.

Current status

I've launched a new version of my portfolio in July. For this version, I wanted to play around with interactions and layout.

I've rebuilt the current portfolio from scratch using HTML5, CSS5, SASS, Compass, Susy Grid, JQuery, and Javascript.

Toolset

  • Wireframing - Mockingbird, FluidUI
  • Visual Design - Photoshop, Illustrator, Sketch
  • Development - HTML5, CSS3, SASS, Compass, Susy Grid, JQuery, Javascript
  • Testing - BrowserStack, Nu HTML Checker, ChromeVox

About me

I am a web developer and designer based in Wellington, New Zealand. I love all things to do with web layouts and user interface design. In my spare time, I like to eat ice-cream and go swimming.
