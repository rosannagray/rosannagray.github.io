A business unit of the New Zealand Treasury.

Interactive map

Using the mobile-first approach, users can link to country information using buttons on a small viewport, and can have the option of using an interactive map on a larger viewport.

Navigating using the keyboard

Thinking about accessibility, users can navigate the website using their keyboard. For example, when tabbing through, users can have the option of skipping to the main content.

When thinking about visual design, we truncated long pieces of text to create a consistent layout for content.

About the project

The brief

To redesign the website for New Zealand Export Credit Office using their brand collateral.

The challenge

Create a responsive website for the business unit using a legacy content management system.

The work

Client liason was a major part of this project. I also collaborated with developers, graphic designers, and a content strategist to integrate the theme into a content management system so that it matched the brand collateral of the New Zealand Export Credit Office.

The result

This site was launched in 2015. You can see their site at www.nzeco.govt.nz . This link will take you to an external website..

Toolset

  • Wireframing - MyBalsamiq
  • Visual Design - Photoshop, Illustrator
  • Development - HTML5, CSS3, SASS, JQuery, Javascript
  • Content Management System - Plone
  • Frontend Framework - Bootstrap 3.3.2
  • Testing - Litmus, Nu HTML Checker, JAWS
