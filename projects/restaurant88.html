Restaurant 88 Website

A website about vietnamese crusine.

In 2013, I redesigned the website for Restaurant 88, a restaurant providing vietnamese crusine in the heart of Wellington.

About the project

The brief

To redesign the website for Restaurant 88.

The challenge

Create a responsive website for the restaurant using their existing brand.

The work

The work consisted of content collection, wireframing, establishing a colour scheme, creating flat design mockups, front-end development, user testing and rolling out the website.

The result

This website was launched in 2013 and is currently still in use. Please note that I no longer maintain their website, but you can visit Restaurant 88's website at restaurant88.co.nz . This link will take you to an external website..

Toolset

  • Wireframing - Mockingbird
  • Visual Design - Photoshop
  • Development - HTML5, CSS3, SASS, JQuery, Javascript
  • Frontend Framework - Bootstrap v2.3.1
  • Testing - BrowserStack
