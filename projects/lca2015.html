About the project

The brief

To help kick off designs for the Linux Conference Auckland 2015.

The challenge

Create a responsive theme using their brand collateral over a tight timeframe.

The work

The work consisted of wireframing, creating a web styleguide, and responsive prototyping.

The result

Initial designs and prototypes were handed over to the Linux Conference Auckland 2015 team. Since then, they continued to work on the designs and launched the website in 2014 for people to register for the event. It has since been replaced by the 2016 website.

Toolset

  • Wireframing - MyBalsamiq
  • Visual Design - Photoshop
  • Development - HTML5, CSS3, SASS, JQuery, Javascript
  • Frontend Framework - Bootstrap v3.1.1
