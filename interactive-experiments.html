About the Project

This was a thrid year university project for an Ubiquitious Computing course. I've created 3 unique applications designed for the Nexus 7. Developed using Flash Builder 4.6 and Flex 4.6.

Poppin'Corn is an application developed for the Nexus 7 for making digital popcorn. The aim of the app is to pop the corn kernels by touching the screen of the device.

Wind Chimes is an application developed for the Nexus 7 for relaxation and zen, using the device's microphone. Blow into the microphone to hear some wind chimes.

Shake Injury is an application developed for the Nexus 7 for shaking a virtual jester bobble head.
