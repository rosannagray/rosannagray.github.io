My Skills

Web Design

My experience in web design includes skills in wireframing, designing mockups in Photoshop, and development of the website using HTML, CSS, and JavaScript. I've completed the first version of my portfolio using WordPress, and have recently switched to using Twitter Bootstrap to create a more responsive website.

I would like to develop on my skills in web design and development, as this is where I feel my skills are the strongest.

Graphic Design

I am experienced with Photoshop and Illustrator to create interesting graphics. I have used these tools to help create assets for 2D games and websites.

3D Modelling and Animation

I have used Maya and Blender to design compelling 3D models. My skills in this area involve box modelling, texturing models, rigging and animation.

Programming

I have experience with Object-Oriented Programming, which is used in the design field to create complex graphics in Processing, and accessing the inputs of mobile and tablet devices to create exciting interactive experiments.
